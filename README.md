# Idea Support Hub

A community-driven platform where innovation meets support. Share your creative ideas and help others bring theirs to life.

## Features

- Submit Ideas - Share your innovative concepts with the community
- Browse Ideas - Explore all submitted ideas in one place
- Support Others - Vote for ideas you believe in
- Track Popularity - See which ideas get the most support
- Persistent Storage - All data saved locally with SQLite

## Installation

1. Clone the repository
git clone https://github.com/Arashmazlumi/idea-support-hub.git
cd idea-support-hub

2. Install dependencies
pip install -r requirements.txt

3. Run the application
streamlit run app.py

4. Open your browser
The app will automatically open at http://localhost:8501

## How to Use

Submit an Idea:
1. Enter a title for your idea
2. Write a description explaining the concept
3. Click Submit Idea button

Support an Idea:
1. Browse through existing ideas
2. Click Support button under any idea
3. Watch the support count increase

## Tech Stack

- Streamlit - Web framework for the UI
- SQLite - Local database for data storage
- Python - Backend logic
